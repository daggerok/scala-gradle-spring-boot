scala-gradle-spring-boot [![build](https://travis-ci.org/daggerok/scala-gradle-spring-boot.svg?branch=master)](https://travis-ci.org/daggerok/scala-gradle-spring-boot)

```shell

open htpp://localhost:8080
```
